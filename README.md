# E33EPUS's ChatScreen

> 100% AI-generated by Claude Code

A mod that replaces the vanilla chat screen with chat bubbles, QQ-style time separators, right-click copy/quote, @mentions, a notification bar, and strong hints. The HUD overlay shows a chat icon with keybind, red dot for unread, message previews, and strong hint popups for system messages and @/quote mentions. Features a customizable title bar, slide-in/out animation, command suggestions, chat history, and clickable links. Config options: mod toggle, red dot, animation, system strong hint, mention strong hint, message preview toggle/lines/width, and own/other bubble and text colors.

Requires **both client and server** — the server forwards quote and @mention metadata between players, while the client handles all rendering and interaction.
