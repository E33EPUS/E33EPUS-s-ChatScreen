# E33EPUS's ChatScreen

微信/QQ 风格的 Minecraft 聊天界面模组 | QQ/WeChat-style chat screen for Minecraft

**Forge 1.20.1 | Client-side only | v0.1.0-beta**

> 本项目 100% 由 AI（Claude Code）生成。This project is 100% AI-generated by Claude Code.

## 功能 Features

- 聊天气泡 + 玩家头像 + 皮肤
- QQ 风格时间分隔线（同分钟内不重复显示时间）
- 右键菜单：复制消息 / 引用回复
- 左下 HUD：聊天气泡图标 + 键位 + 红点提醒 + 消息预览
- 标题栏可编辑（点击 ✎）
- 打开/关闭滑入滑出动画
- 指令补全支持
- 聊天历史上下键切换
- 中英文自动切换
- 可配置气泡颜色、文字颜色、红点开关、动画开关

## 安装 Install

1. 安装 Minecraft Forge 1.20.1
2. 下载 [e33chat-0.1.0-beta.jar](https://github.com/E33EPUS/E33EPUS-s-ChatScreen/releases)
3. 放入 `.minecraft/mods/` 文件夹

## 使用方法 Usage

| 操作 | 说明 |
|------|------|
| 打开聊天 | 按聊天键（默认 `T`）或点击左下气泡图标 |
| 发送消息 | 输入后按 Enter 或点击发送图标 |
| 复制消息 | 右键气泡 → 复制 |
| 引用回复 | 右键气泡 → 引用 |
| 编辑标题 | 点击标题旁的 ✎ |
| 设置 | 左下齿轮图标 或 Mods 列表 → 配置 |

## 构建 Build

```bash
./gradlew build
```
