# E33EPUS's ChatScreen

A QQ-style chat overhaul for Minecraft 1.20.1 Forge — bubbles, avatars, time separators, and a slide-in panel that replaces the vanilla HUD. Plays nice with other mods.

**Client + Server** required.

## Highlights

- **Bubbles, not bars** — each message renders as a chat bubble with the sender's player head, QQ/TIM style
- **Zero mod conflicts** — messages pass through all mods before rendering; Xaero waypoints, FTB invites, and future mods all work out of the box
- **Strong hints** — @mentions and important system messages pop a heads-up above the hotbar so you never miss them
- **Right-click menu** — copy any message or start a quote reply with one click
- **Click-to-@** — left-click any avatar to drop `@playername` into the input
- **Fully configurable** — bubble color, text color, background, animations, previews, strong hints, mention highlight — toggle or tweak everything

## Also

- Command suggestions while typing
- Chat history (arrow keys / mouse wheel)
- Per-world custom title bar (click the ✎ to rename)
- Unread notification bar when scrolled up
- Message preview fading above the hotbar

## Install

1. Drop the `.jar` into both client and server `mods/` folders
2. Launch and open chat — the vanilla HUD is replaced automatically
3. Tweak visuals in `config/e33chat-client.toml` (bubble colors, animation, etc.)

# E33EPUS's ChatScreen（中文）

Minecraft 1.20.1 Forge 的 QQ 风格聊天重制——聊天气泡、头像、时间分隔、侧滑面板替代原版聊天框，与其他 mod 无冲突。

需要**客户端+服务端**同时安装。

## 核心亮点

- **聊天气泡**——每条消息渲染为带发送者头像的气泡，QQ/TIM 风格
- **零 mod 冲突**——消息先走完所有 mod 的管线再渲染；Xaero 路径点、FTB 邀请、未来新 mod 全部开箱即用
- **强提示**——被 @或收到重要系统消息时，快捷栏上方弹出醒目提示，不会错过
- **右键菜单**——一键复制消息或引用回复
- **点头像即 @**——左击任意头像自动填入 `@玩家名`
- **全可配置**——气泡颜色、文字颜色、背景、动画、消息预览、强提示、@高亮，想关就关想调就调

## 其他功能

- 输入时指令补全
- 聊天历史（方向键 / 滚轮）
- 每个世界独立标题栏（点 ✎ 改名）
- 向上滚动时未读提示条
- 快捷栏上方消息预览淡入淡出

## 安装

1. 把 `.jar` 丢进客户端和服务端的 `mods/` 文件夹
2. 启动游戏，打开聊天——原版聊天框自动替换
3. 在 `config/e33chat-client.toml` 调整外观（气泡颜色、动画等）
